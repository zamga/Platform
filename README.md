# ApexLedger Financial Website

A premium, responsive financial services landing page built with static HTML, CSS, and JavaScript.

## Files

- `index.html` - Page structure and finance-focused content.
- `styles.css` - Responsive visual system, layout, and component styling.
- `script.js` - Mobile navigation, scroll reveal animation, KPI counters, and ROI calculator behavior.

## Preview locally

Open `index.html` in a browser or serve the folder with any static file server.
