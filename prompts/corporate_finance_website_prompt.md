# Prompts for Creating a Top-Tier, Award-Winning Corporate Finance Website

This repository contains highly optimized, hyper-detailed visual and design prompts tailored specifically for **Flowstep** and **Google Stitch (formerly Galileo AI)** to generate a world-class, award-winning corporate finance website.

---

## 1. The Core Design Vision & Aesthetic Direction

To win design awards (Awwwards, CSS Design Awards, FWA) in corporate finance, the aesthetic must transcend boring corporate templates and enter **"vibe design" excellence**:

*   **Aesthetic:** Ultra-premium, hyper-modern, minimalist-futuristic (high-end institutional meets high-tech).
*   **Colors:** Deep Obsidian Black (#0A0A0C) or extremely dark slate base with deep metallic accents: liquid-metal chrome/platinum, refined warm gold/bronze, or vibrant cyan-blue electric gradients for interactive hotspots. 
*   **Typography:** Bold, high-contrast, editorial grotesque typography. Huge header fonts with tight tracking paired with clean, geometric, high-legibility sans-serif body fonts.
*   **Visual Elements:** Premium 3D glassmorphism, interactive bento-box layouts, ultra-clean financial data visualizations, custom abstract liquid-metal motion graphics, and sleek 3D architectural representations.
*   **Atmosphere:** Authoritative, high-trust, elite, sleek, and high-tech.

---

## 2. Master Flowstep Prompt (Web Page UI Design)

Flowstep generates UI screens and converts them into production-ready React/Tailwind/TypeScript code. Use this highly detailed, visual, structure-focused prompt.

```text
Create a world-class, award-winning corporate finance & institutional investment website home screen. The aesthetic is ultra-premium, dark-mode, and hyper-modern, combining absolute financial authority with futuristic tech.

[LAYOUT & STRUCTURE]
- Header: Minimalist navigation with a subtle glowing borders, logo "NEXUS CAPITAL" in bold tracking sans-serif, standard links (Solutions, Portfolio, Intelligence, Company), and a high-contrast pill-shaped "Institutional Portal" action button.
- Hero Section: Large, eye-catching bold editorial header ("THE FUTURE OF CAPITAL, ALIGNED WITH PERSPECTIVE") using high-contrast typography. Subtitle below in light gray explaining the firm's global quantitative approach. Place a refined call-to-action button in platinum glassmorphism with a glowing border.
- Interactive Dashboard Preview: Below the hero, show an interactive 3D mock financial interface with gorgeous glowing charts, live asset allocation circles (glassmorphism charts in dark neon teal and gold), and portfolio health indices.
- Feature Grid (Bento Box style): A multi-column grid with dark-themed cards highlighting key solutions:
  1. "Quantitative Execution" showing a sleek abstract line-graph overlay.
  2. "Algorithmic Risk Assessment" featuring custom visual nodes.
  3. "Global Liquidity Access" featuring a glowing digital globe representation.
- Metrics & Proof Points: Huge numbers (e.g., "$42.8B AUM" and "99.9% Uptime") in gold or polished silver chrome, with tiny elegant labels beneath.
- Footer: Deep corporate footer with multi-column sitemap, legal compliance small-text, and a subtle email subscription input using thin gray borders and glowing focus states.

[VISUAL STYLE & COLORS]
- Color Palette: Main background #070709 (Obsidian Black). Primary accents in liquid platinum, refined champaign gold, and deep glowing neon cyan (#00F2FE) for data lines.
- Glassmorphism: Cards must have a 5% translucent white background with a highly polished backdrop-blur (blur-xl) and 1px borders of 10% white/gold opacity.
- Shadows & Glows: Incorporate modern ambient drop-shadows and subtle colorful back-glows (radial gradients) behind dashboard panels to simulate depth and motion.
- Styling: Strict modern CSS/Tailwind vibes, featuring perfectly rounded corners (rounded-2xl) and precise spacing. Ensure there are no boring standard generic web templates.
```

## 3. Master Google Stitch Prompt (Multi-Platform / Standard & Experimental)

Google Stitch specializes in transforming sketches or detailed structural prompts into structured components. This prompt is structured specifically to make the AI generate rich visual components.

```text
Design a responsive, award-winning web dashboard landing page for a cutting-edge global corporate finance and wealth advisory firm.

[THEME & MOOD]
An premium corporate finance aesthetic with high contrast and elite technical feel. Use a midnight dark background theme with refined neon details.

[VISUAL SYSTEM]
- Primary Background: #0B0B0F (rich deep charcoal obsidian)
- Secondary Background (Cards): #13131A with solid 1px borders of #21212B and backdrop filter blur.
- Primary Accent: Brilliant electric sapphire (#1F51FF) and polished metallic silver (#E5E8EB) to emphasize security, trust, and intelligence.
- Typography: Heavy, ultra-modern sans-serif (such as Inter or SF Pro Display) with wide letter spacing for subtitles and high contrast for major figures.

[HERO COMPONENT]
- Left column: Bold, editorial typography saying "Redefining Sovereign Capital". Beneath it, a refined description: "We blend quantitative machine learning models with elite human expertise to scale global assets." Include two clean glass-morphic buttons ("Explore Funds" and "Partner With Us").
- Right column: A gorgeous interactive dashboard panel showing a real-time portfolio allocation donut chart with sleek glowing segments (teal, silver, and gold), absolute performance index percentages, and a 3D-styled geometric grid line pattern overlay.

[PORTFOLIO VALUE COMPONENT]
- A beautiful bento-box showcase grid.
- Top Left Card: "AUM Performance" displaying a glowing neon line chart with elegant hover dots.
- Top Right Card: "Direct Institutional Portal" with a premium dark security key icon and minimalist input fields.
- Bottom Full-Width Card: "Global Ventures Mapping" demonstrating a futuristic network graph visualization of international transaction flows.

[FOOTER COMPONENT]
- Clean, multi-column navigation, privacy policy agreements, compliance statements in refined slate-gray typography, and an interactive dark newsletter signup field.
```

## 4. Complementary Refining & Iterative Prompts

Once the initial screen is generated in Flowstep or Stitch, copy and paste these follow-up prompts to polish the website into an award winner.

### Prompt A: Refining the Visual Polish (Vibe Design Upgrade)
> *"Make the visual borders and lines incredibly clean. Make the card borders 1px solid with 10% opacity white, mimicking premium glassmorphism. Add a soft glowing neon cyan radial gradient behind the main dashboard component to create an immersive sense of 3D depth. Increase typography contrast by making the main headers pure white (#FFFFFF) and secondary text a highly readable warm gray (#94A3B8)."*

### Prompt B: Adding Elite Financial Data Visualizations
> *"Replace the standard column charts with an extremely elegant custom line graph of market volatility. The line should have a neon electric-blue glow and a translucent color gradient fill stretching down to the bottom of the card. Add tiny, ultra-precise coordinate numbers and indicators on the axes to represent sophisticated quantitative data."*

### Prompt C: Optimizing Mobile View (For Google Stitch)
> *"Convert this dashboard page into a premium mobile-responsive view. Stacking the columns beautifully, change the bento grid cards into a swipeable carousel of glassmorphic financial metric cards, and turn the main navigation header into an elegant, slide-out full-screen overlay menu with gold-tinted accents."*
